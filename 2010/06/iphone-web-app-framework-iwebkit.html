iPhone Web App Framework - iWebKit

Jun 11th, 2010 | Comments

iWebKit  是專為 Apple 的行動裝置而設的一套 Framework, iWebKit 提供的介面和 iPhone 內置的 App 介面非常相似，可以令普通使用者以為是一個 iPhone App 而不是一個網頁，簡單又易用，推介給需要製作 iPhone 或 iPad Web App 的朋友。
