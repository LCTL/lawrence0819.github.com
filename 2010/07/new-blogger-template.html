換了新的 Blogger 模版

Jul 6th, 2010 | Comments

終於換了新的 Blogger 模版了，舊的模版令我用不到 Yahoo! Site Explorer，驗證網站時 Blogger 回傳 HTTP Error 500 Internal Server Error 這個訊息。舊的模版是使用 Blogger Template Designer 改裝而成的，用 Blogger Template Designer 製作出來的模版會根據不同的瀏覽器自動作出微調的，應該就是這個原因令 Yahoo! Site Explorer 的程式不能順利找到 y_key。我不知道這個原因會不會導致 Yahoo 搜尋引擎的 bot 也不能順利進入頁面提取資料。所以還是換掉舊模版比較安心。

換新模版的同時，我對這個 Blogger 的 Javascript 進行了優化，令到載入頁面的時間大大降低，也將 ShadowBox 換成比較輕巧一些的 FancyBox。最下面的工具列也一併移除掉來換取更快的速度。
