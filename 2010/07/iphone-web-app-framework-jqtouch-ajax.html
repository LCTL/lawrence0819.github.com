iPhone Web App Framework - jQTouch

Jul 22nd, 2010 | Comments

發現了一款 Open Source 的工具，可以用來開發 iPhone Web App。

jQTouch 是一款基於 jQuery Javascript Framework 開發給 iPhone / iPad / iPoad Touch 使用的一套 Framework，擁有比 iWebKit 更華麗的特效，亦可以自由去替換各種主題 (Theme)，因為是建基於 jQuery，因此可以自由加入插件 (Extensions)。jQTouch 亦已經處理了一些只有在手機裝置 (Mobile Device) 上或觸摸屏 (Touch Screen) 上發生的事件 (Event)。和 iWebKit 最大分別的地方是使用了 AJAX 技術去拿取資料，即是說你的瀏覽器不會去跳頁，這樣可以增加瀏覽速度，令用戶有更好的瀏覽體驗。
